Niconico autologin extension
======================
Niconico autologin extensionはニコニコ動画のログイン処理を自動化するChromeエクステンションです。
Chromeエクステンションの設定画面から、ニコニコ動画のアカウント情報を保存すると、ニコニコ動画のログイン画面で自動的にその情報を送信します。
自分でログインボタンを押さなくてもよくなり、1クリックする手間が無くなります。
 
使い方
------
Niconico autologin extensionをインストールした後、Chromeのメニューから「環境設定」を選択し、設定画面を開きます。

左のメニューから「機能拡張」を選択し、「Niconico autologin extension」の項目の「オプション」を選択します。

すると、「Niconico autologin extension」の設定画面が開くので、メールアドレスとパスワードを入力し保存ボタンを選択すると設定完了です。

今後はニコニコ動画のログイン画面が表示されると、自動でログイン処理が行われます。
